CIDADÃO ITALIANO
CONTEXTO DO PROJETO
Projeto de identidade visual para serviço. A empresa em questão se chama Cidadão Italiano e tem como principal objetivo auxiliar brasileiros a conseguir a cidadania italiana e migrar de país. Uma breve contextualização… Nesse projeto desenvolvido a uma micro empresa, para qual prestava serviço como designer, fui responsável por desenvolver a logomarca. Veja na imagem abaixo o resultado desse trabalho.

PROCESSO
Esta é a prancha principal onde foram feitos alguns dos sketchs para este projeto. inúmeras ideias foram sugeridas ou pelos responsáveis da empresa ou por participantes do projeto na equipe onde trabalhava, mas havia apenas uma exigência, o projeto deveria conter as cores da bandeira italiana.


O primeiro quadro abaixo (a esquerda) respectivamente foi o primeiro sketch, focado em trazer as cores e a informação “nome”. O segundo é uma definição de uma ideia simbólica. Como é possível ver na prancha principal em meio a inúmeros traços soltos, o principal ícone utilizado para representar os serviços prestados pela Cidadão Italiano é o passaporte.

Os quadros abaixo são a soma das principais ideias geradas durante o desenvolvimento desse trabalho.

RESULTADO
Vetorização e finalização do trabalho: Os principais princípios seguidos no desenvolvimento desse tipo de trabalho sem dúvidas são desenvolver um símbolo icônico que sirva para representar muito bem o cliente de maneira inconfundível, que seja atemporal e que possa ter um emprego prático independente do meio ou material utilizado, seja impresso, monitor ou tela de celular. abaixo o resultado do trabalho.
