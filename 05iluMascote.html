JOGADOR BOM DE BOLA F.C.
CONTEXTO DO PROJETO
Este é um mascote feito para um time de futebol amador.

PROCESSO
O processo de desenvolvimento para esse projeto foi claro e objetivo desde o principio, o cliente sabia muito bem o que procurava. Foram feitos alguns sketchs explorando formas, design de personagens e cores.

RESULTADO
A arte desenvolvida no projeto foi usada para produzir troféus para o campeonato do time e divulgação nas redes sociais.
